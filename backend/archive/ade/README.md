# ADE — AITDL Development Engine

The ADE orchestrates the full software development lifecycle within the AITDL ecosystem.

## ADE Workflow (Rule 5)

```
1. Understand the request
2. Analyze project architecture
3. Generate modules properly
4. Implement minimal changes
5. Add tests
6. Update documentation
7. Pass Guardian validation
```

## Multi-Agent Roles (Rule 6)

| Agent | Responsibility |
|---|---|
| **Architect Agent** | Plans architecture |
| **Coder Agent** | Writes code |
| **Security Agent** | Validates security |
| **Test Agent** | Generates tests |
| **Documentation Agent** | Updates docs |

## Files (Phase 2+)

| File | Purpose |
|---|---|
| `ade/workflow.json` | ADE step definitions |
| `ade/orchestrator.py` | Agent coordination logic |
| `ade/context.md` | Current development context |
