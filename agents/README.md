# Agents — AITDL Multi-Agent System

This directory contains AI agent definitions and configurations for the AITDL ecosystem.

## Agent Roles (Rule 6)

Each agent has a single responsibility and communicates through the ADE orchestrator.

| Agent | File | Role |
|---|---|---|
| Architect Agent | `agents/architect.md` | Plans architecture and module design |
| Coder Agent | `agents/coder.md` | Writes and refactors code |
| Security Agent | `agents/security.md` | Validates security and CORS |
| Test Agent | `agents/test.md` | Generates test suites |
| Documentation Agent | `agents/docs.md` | Keeps docs current |

## Constraints

- All agents respect RULESBOOK Rules 1–15
- No agent may bypass Guardian validation
- No agent may create unauthorized root directories
